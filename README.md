The Vehicle Insurance System is a Java-based desktop application that automates premium calculation and management for cars, bikes, and trucks. Built using OOP concepts, it ensures modularity and reusability. A Swing GUI enables easy input and tracking, while InsuranceRecords.txt stores data to access and add records. Premiums are calculated as: Car—5% value − ₹100 per airbag; Bike—3% value + ₹200 (ABS); Truck—6% value + ₹10 per ton.
